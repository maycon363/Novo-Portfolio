Jarvis IA (Full Stack + 3D)

Desafio: Desenvolver um sistema de IA de conversação com uma interface 3D altamente otimizada e comunicação em tempo real.
Solução: O Frontend (React Three Fiber) gerencia o modelo do Homem de Ferro. Implementei PBR (Physically Based Rendering) para materiais realistas e controle dinâmico de emissão/luz, criando um efeito de pulso em tempo real quando a IA está "falando". Foi utilizado A API da Groq. O Backend (Node/Express/Socket.io) garante a troca de mensagens e o histórico com MongoDB(Local).

React Three Fiber(3D) TypeScript Vite CSS3 PBR Materials Otimização Mobile Socket.io(Real-Time) Node.js/MongoDB
